Tutorial de Uso da API - Estacionamento

Tecnologias Utilizadas

  • Framework: Laravel 9
  • Linguagem de Programação: PHP
  • Banco de Dados: MySQL (conectado via XAMPP)

1. Criar uma vaga no estacionamento

Método: POST
URL: 127.0.0.1:8000/api/vagas 
{
  "tipo": "comum", 
  "status": "disponivel"
}

2. Listar as vagas

Método: GET
URL: 127.0.0.1:8000/api/vagas 
{}

3. Listar uma vaga específica

Método: GET
URL: 127.0.0.1:8000/api/vagas/{id} 
{}

4. Deletar uma vaga do estacionamento

Método: DELETE
URL: 127.0.0.1:8000/api/vagas/{id} 
{}

5. Ocupar uma vaga

Método: POST
URL: 127.0.0.1:8000/api/vaga-ocupada/{id}/ocupar 
{
    "placa": "ABC1323"
}

6. Desocupar uma vaga

Método: PUT
URL: 127.0.0.1:8000/api/vaga-ocupada/{id}/desocupar 
{}

7. Alterar o tipo de uma vaga

Método: PUT
URL: 127.0.0.1:8000/api/vagas/{id}/tipo 
{
  "tipo": "deficiente"
}

8. Listar pagamentos

Método: GET
URL: 127.0.0.1:8000/api/pagamentos/ 
{}

9. Realizar pagamento

Método: PUT
URL: 127.0.0.1:8000/api/pagamentos/{id}/pagar 
{}
